# Get-id
获取未注册的短位 GitHub ID 

## 环境安装
```bash
# Install python pip

pip install --upgrade pip && pip install -r requirements.txt
```

## 校验数据
```bash 
# Run

python -u review.py
```
## 匹配查找
```bash 
# Run

python -u find.py
```

## 重新获取
```bash
# Run

python -u run.py
```
